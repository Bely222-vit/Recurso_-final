Conversión de sistema binario a decimal

El proceso para convertir un número del sistema binario al decimal, es que cada uno de los dígitos se multiplican por las potencias de dos iniciado con la potencia de 0, es decir (20, 21, 22 , 23… etc.), comenzando por el LSB (Bit Menos Significativo) cuyo exponente es 0 en el bit situado más a la derecha hasta alcanzar al MSB (Bit Más Significativo) cuyo exponente dependerá de su posición hacia la izquierda. Después de realizar cada una de las multiplicaciones, se debe sumar el resultado de todas y él número resultante será el equivalente al Sistema Decimal.

Por ejemplo, para convertir el número binario 110010112 a decimal, lo desarrollamos teniendo en cuenta el valor de cada bit:

Una vez entendido el valor de cada bit, se procede a realizar las respectivas multiplicaciones, para luego sumar esos resultado y obtener el resultado en decimal:

1 x 27 + 1 x 26 + 0 x 25 + 0 x 24 + 1 x 23 + 0 x 22 + 1 x 21 + 1 x 20 = 128 + 64 + 8 + 2 + 1 = 20310

Por ejemplo, para convertir el número binario 10100112 a decimal, lo desarrollamos teniendo en cuenta el valor de cada bit:

1 x 26 + 0 x 25 + 1 x 24 + 0 x 23 + 0 x 22 + 1 x 21 + 1 x 20 = 83

10100112 = 8310

(Gómez y Mojica, 2014)
