Conversión de sistema octal a decimal

El proceso para convertir un número del sistema octal a decimal,  cada uno de los dígitos se multiplican por la base 8 iniciado con la potencia de 0, es decir (80 , 81 , 82 , 83 … etc.), comenzando por el LSB (Bit Menos Significativo) cuyo exponente es 0 en el bit situado más a la derecha hasta alcanzar al MSB (Bit Más Significativo) cuyo exponente dependerá de su posición hacia la izquierda. Después de realizar cada una de las multiplicaciones, se debe sumar el resultado de todas y él número resultante será el equivalente al Sistema Decimal.

Por ejemplo, para convertir el número 2378 a decimal basta con desarrollar el valor de cada dígito:

2 x 82 + 3 x 81 + 7 x 80 = 128 + 24 + 7 = 159
2378 = 15910

(Universidad Don Bosco, 2020)

Para convertir el número octal 2348 a decimal, cada uno de los dígitos se multiplica por las potencias de la base 8. Completa el cálculo siguiente:

2 x 82 + 3 x 81 + 4 x 80 = JXUwMDY5JXUwMDAzJXUwMDBh  + JXUwMDZhJXUwMDA2  + JXUwMDZj  = JXUwMDY5JXUwMDA0JXUwMDAz
