Conversión entre Sistemas de Numeración

Conversión de Decimales a Binarios

Para convertir un número decimal al sistema binario basta con realizar divisiones sucesivas por 2, se divide de manera entera al número proporcionado entre dos, obteniendo un cociente y un residuo. Si el cociente no es 1, este se vuelve a dividir entre 2. Si el nuevo cociente no es 1, se vuelve a dividir entre 2. El proceso anterior se repite sucesivamente hasta que el cociente sea 1.

Finalmente, se escriben los restos obtenidos en cada división en orden inverso al que han sido obtenidos, ya que estos nos proporcionan el número inicial expresado en el sistema binario.
